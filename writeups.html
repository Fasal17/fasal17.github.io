Security Write-Ups & Reports

Cross-Site Scripting (XSS) Vulnerability

Vulnerability Type: Cross-Site Scripting (XSS)

Severity: Low

An XSS issue found in the site's comment section. User-supplied input was not sanitized, allowing script injection.

Details: In the bug bounty program, I discovered that comment inputs were not escaped. By injecting a simple <script> alert command, I could execute arbitrary JavaScript. Steps to reproduce and remediation are documented in the full write-up.

SQL Injection in Login Form

Vulnerability Type: SQL Injection

Severity: High

The login form was vulnerable to SQL injection, allowing attackers to bypass authentication.

Details: During a penetration test, input fields were tested and found susceptible to SQL queries. I demonstrated access to the admin account by using classic SQL injection payloads. Detailed findings and fixes are in the report.

Server Configuration Mistake

Vulnerability Type: Insecure Configuration

Severity: Medium

A misconfigured server allowed directory listing, exposing sensitive files.

Details: During a security lab, the web server was found listing directory contents due to a missing index page. I documented how this allowed access to backup files, and recommended disabling directory browsing.
